Wist
Yet another modern Tetris clone, mostly taking inspirations from TETR.IO

  • Programmer: Link

  • Contributor: TwoQuantumBits - Design, User Interface, & User Experience

Features
  • ✅ Lock
  • ✅ Move reset
  • ✅ 7 Bag
  • ✅ Next 5 Pieces
  • ✅ Alert when in danger (everything turns red, marking where the next piece with spawn with ❌, showing an exclaimation mark image if the current piece will cause top out, in the case of multiplayer, also taking garbage into account)
  • ✅ Wall kicks:
    • ✅ SRS+
      • ✅ SRS
      • ✅ TETR.IO SRS+ wall kicks
    • ✅ TETR.IO 180 wall kicks
  • ✅ Singleplayer:
    • ✅ Marathon
    • ✅ 40 Lines
    • ✅ Ultra
  • ✅ Configurable handling
    • ✅ DAS
    • ✅ ARR
    • ✅ ARR, 0 ARR
    • ✅ SDF, ∞ SDF
  • ✅ Configurable controls:
    • ✅ Move falling piece left
    • ✅ Move falling piece right
    • ✅ Soft drop
    • ✅ Hard drop
    • ✅ Wist 90
    • ✅ Wist 180
    • ✅ Wist 270
    • ✅ Hold
  • ✅ Multiplayer
    • ✅ Combo Table:
      • ✅ TETR.IO Combo Table
      • ✅ Attack type: Single - Garbage sent at combo x: floor(sqrt(x))
    • ✅ Send and receive garbage
    • ✅ Combo blocking
    • ✅ Garbage is separately “tanked” (accepted), therefore nerfing 4 wide
    • ✅ Same bag for both players
  • ✅ TETR.IO Season 2
    • ✅ Back-to-back charging
    • ✅ All-mini
    • ✅ All clear nerf
    • ✅ Opener phase nerf
    • ✅ Quads and Spins (including All-Spins) that clear garbage now send a flat (not influenced by multipliers) +1 extra garbage.
  • Status: Farewell
  • Source Code: https://git.sr.ht/~lts20050703/wist/
  • License: Affero GNU Public License version 3
